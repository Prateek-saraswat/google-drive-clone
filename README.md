# Google Drive Clone (Minor Project)

This is a lightweight Google Drive clone built as part of a minor project using Node.js (HTTP module) and vanilla frontend technologies. It implements core file management features such as uploading, viewing, renaming, downloading with progress tracking, and file preview â€” all without using any external frameworks like Express.

---

## Features

- File upload with progress indicator
- View list of uploaded files
- Preview files (e.g., images)
- Rename existing files
- Download files with progress tracking
- Simple and clean UI using HTML, CSS, and JavaScript

---

## Technologies Used

- Node.js (HTTP and File System modules)
- mime types
- HTML5
- CSS3
- JavaScript
- React js
